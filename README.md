#  地域イベントアプリ

##  概要

地域のイベント情報を一覧・検索できるWebアプリです。
ユーザーが気軽にイベントを見つけられることを目的としています。

---

##  セットアップ手順

```bash
git clone （https://github.com/1234risa/event_app）
cd （event_app）
npm install
npm start
```

ブラウザで以下にアクセス👇
http://localhost:3000

---

##  使用技術・ライブラリ

* React
* JavaScript (ES6)
* CSS
* react-calendar
* Font Awesome

---

##  実装機能

*  イベント一覧表示
*  ジャンルフィルター機能
*  カレンダー表示
*  メモ機能
*  コンポーネント分割

---

## 🔍 動作確認方法

1. アプリを起動
2. イベント一覧が表示されることを確認
3. ジャンルで絞り込みができることを確認
4. カレンダーの日付選択で表示が変わることを確認
5. メモ機能が使えることを確認
